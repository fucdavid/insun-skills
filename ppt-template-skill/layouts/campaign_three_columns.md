# campaign_three_columns

## 适用场景

用于三栏运营/营销规划页，例如 KOC/KOL 任务包、节日营销、三大活动专题、三条传播主线。

## 页面结构

- 顶部：一句总论/发力方向。
- 中部：三列专题卡。
- 每列包含：场景说明、专题条、活动标题、任务说明、两个方向/动作、两个素材图位、目标指标。
- 底部：机制强化或协同方式总结。

## 字段

```json
{
  "layout": "campaign_three_columns",
  "headline": "三大重点事项发力，合力凸显品牌调性",
  "columns": [
    {
      "scenario": "结合用车场景凸显体验",
      "tag": "五一出行专题",
      "title": "#五一出行季",
      "desc": "聚焦用车场景，邀请 KOC 分享真实体验。",
      "directions": [
        { "label": "方向一：挑战测评类" },
        { "label": "方向二：体验分享" }
      ],
      "metrics": "计划邀约XX名用户，产出XX条内容"
    }
  ],
  "footer": "任务发布邀约 + 优质创作者1V1邀约 + 公域创作主动邀约"
}
```

## 限制

- 必须固定 3 栏，少于或多于 3 栏都不要使用。
- `headline` 控制在 36 字以内。
- 每列 `scenario` 控制在 14 字以内。
- 每列 `tag` 控制在 12 字以内。
- 每列 `desc` 控制在 55 字以内。
- 每列 `metrics` 控制在 28 字以内。
- 如果有真实图片素材，优先填入素材图位；没有图片时保留浅色素材占位，不使用随机网络图。
- 专题条颜色默认使用当前模板 `primary_color` / `accent_color`，参考图颜色不能跨模板照搬。
