# insight_onepage

## 适用场景

用于“观点 + 逻辑路径 + 数据证明 + 金句”的单页论证型内容，例如行业趋势、战略判断、方法论开场。

## 页面结构

- 顶部：内容页标题和可选副标题，优先使用模板原有标题文本框。
- 左上：核心观点。
- 左下：3-5 个演进路径、阶段或逻辑链节点。
- 右侧：2-4 个关键数据卡片。
- 右下：一句结论金句。

## 字段

```json
{
  "layout": "insight_onepage",
  "insight": "核心观点",
  "path_title": "关键演进路径",
  "path": [
    { "value": "1.0", "label": "产品驱动", "desc": "代表案例" }
  ],
  "data_title": "关键数据",
  "data": [
    { "value": "54.9", "label": "行业 NPS", "desc": "创历史新低" }
  ],
  "quote": "过去，品牌经营产品；现在，品牌必须经营用户关系。"
}
```

## 限制

- `insight` 控制在 45 字以内。
- `path` 最多 5 个节点。
- `data` 最多 4 个卡片。
- `quote` 控制在 28 字以内。
- 配色必须继承当前模板 `design_tokens`。
