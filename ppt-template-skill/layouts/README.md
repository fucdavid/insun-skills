# 排版库使用说明

本目录用于登记内容页排版库，遵循渐进式披露原则。

## 使用流程

1. 先读取 `registry.json`，根据内容类型选择候选布局。
2. 确定布局后，只读取该布局对应的详情文件。
3. 生成 `content.json` 时，只填写当前布局需要的字段。
4. 不要一次加载所有布局详情，也不要在同一页混塞多个布局字段。

## 新增布局要求

新增布局必须同时完成：

- 在 `registry.json` 登记布局名、别名、适用内容类型、核心字段和限制摘要。
- 新增一个独立 `.md` 详情文件，说明页面结构、字段 schema、字数限制和示例 JSON。
- 在 `scripts/render_geely_report.py` 中实现或登记可用的布局函数。
- 至少用一个已登记模板生成样例验证。

## 命名规范

- 使用语义化名称，例如 `campaign_three_columns`、`insight_onepage`。
- 不使用品牌名、车型名、项目名或截图文件名。
- 变体使用后缀：`_compact`、`_visual`、`_text`。
