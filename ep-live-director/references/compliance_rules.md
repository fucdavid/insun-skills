## 一、绝对禁用词（极限词）

以下词汇**在任何场景下均不得使用**，一经检测立即替换为对应的"合规平替词"：

| 禁用词 | 合规平替词 | 使用场景参考 |
|--------|-----------|--------------|
| 最好 | 表现突出 / 硬核实力 / 备受认可 | 性能描述 |
| 最强 | 行业前列 / 出色的 / 强悍的 | 性能/配置描述 |
| 第一 | 行业前列 / 率先搭载 / 领先布局 | 技术/销量描述 |
| 唯一 | 罕见 / 少有竞品能做到的 / 差异化配置 | 功能独特性描述 |
| 顶级 | 旗舰级 / 高阶 / 领先水准 | 配置定级描述 |
| No.1 | 销量领先 / 市场占有率领先 / 热销车型 | 销量/榜单描述 |
| 绝对 | 可以说 / 在实测中表现为 / 数据显示 | 性能描述 |
| 超越一切 | 在同级别中表现亮眼 / 综合实力突出 | 竞品比较 |
| 无敌 | 在同价位段表现优异 / 极具竞争力 | 价值描述 |
| 史上 | 目前 / 迄今 / 此次 | 时间范围描述 |
| 永久 | 长期 / 持续提供 / 后续政策以门店为准 | 服务/政策描述 |
| 保证 | 根据历史数据表现 / 在标准测试条件下 | 性能承诺 |
| 秒杀 | 优于 / 高于 / 在对比测试中领先 | 竞品对比 |
| 吊打 | 在同价位段形成明显优势 | 竞品对比 |

## 二、条件限制用语规范

涉及以下场景时，**必须附加限制说明**，不得省略：

| 场景 | 必须附加的限制语 |
|------|----------------|
| 续航里程 | 「续航数据来源于工况测试，实际续航因驾驶习惯、路况、温度等因素有所差异」 |
| 充电速度 | 「充电时间以满足充电条件的场景为准，实际数据以车辆实时显示为准」 |
| 购车活动/优惠政策 | 「活动政策以线下门店实际情况为准，请咨询当地授权经销商」 |
| 智驾辅助功能 | 「智驾辅助系统为辅助驾驶功能，不能替代驾驶员注意力，使用时请保持对路况的关注」 |
| 保险/金融分期 | 「金融方案以合作机构实际审批为准，详情咨询门店顾问」 |
| 碰撞安全成绩 | 「碰撞测试成绩来源于[机构名称]标准测试条件，实际安全表现受多因素影响」 |

## 三、车型配置混用防错表

### eπ007（五座版）——禁止提及
- ❌ 三排座椅
- ❌ 6/7座布局
- ❌ 三排过道/穿越通道
- ❌ 第三排头部空间

### eπ008（五座版）——禁止提及
- ❌ 三排座椅
- ❌ 三排过道
- ❌ 7座布局相关描述

### eπ008（六座版）——可以提及
- ✅ 三排座椅（第三排空间需如实描述，不可夸大）
- ✅ 独立座椅布局（六座版为2+2+2布局）
- ✅ 三排进出便利性

> **操作规范**：开始撰写前，必须先确认车型版本，在文稿顶部用注释标注，全程统一。

## 四、竞品对比合规规范

- **禁止直接点名竞品车型**做负面比较，如「比XX车强太多了」
- **可以使用**：「同级别车型中」「同价位段」「对比我们自己之前的版本」
- 如需数据对比，必须注明数据来源与测试标准
- 禁止使用「完爆」「碾压」「吊打」等贬损性词汇

## 五、直播常用合规表达模板

```
【技术优势表达】
✅ 「eπ008搭载了行业前列的XXX技术，在实际测试中，百公里加速可以达到X.X秒。」
❌ 「eπ008是目前最强的新能源车！加速绝对天下第一！」

【续航描述表达】
✅ 「在CLTC标准工况测试下，纯电续航可达XXX公里，综合工况下实测通常在XXX公里上下，具体数据因实际驾驶情况而异。」
❌ 「续航XXX公里，绝对够用，永远不焦虑！」

【活动政策表达】
✅ 「现在下订的朋友，可以享受到XXX元的购车优惠，具体政策以线下门店实际情况为准，大家可以扫码或联系当地门店确认。」
❌ 「今天下订保证享受XXX优惠，这个政策唯一！」

【智驾功能表达】
✅ 「在开启高速智驾辅助功能后，车辆可以自动完成跟车、超车等操作，但需要驾驶员全程保持注意力，智驾是辅助系统，不是自动驾驶。」
❌ 「开了这个功能，手可以离方向盘，绝对安全！」
```

## 六、弹幕互动合规建议

- 回复用户关于「保值率」的问题时，避免承诺具体数字，使用「根据市场行情参考」
- 不得在直播中公布未经授权的内部优惠信息
- 礼品抽奖不得暗示「必中」，使用「参与即有机会」的表达
- 所有承诺型话术（如赠品、延保）需加注「以实际交付为准」
